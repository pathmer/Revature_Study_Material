# Java Day 7

### Exceptions

- Exceptions are Objects!
-  'Thrown' whenever code executes along an unhappy path 
- Two kinds of Exceptions:
    - Checked Exceptions
        - Compile time
        - must be handled for code to compile
    - Unchecked Exceptions
        - Runtime
        - Are not required to be handled
            - but you should...

**Keywords**  
- Try
- Catch
- Throw
- Finally
- Throws
